# Add Candidate Workflow

## Overview

The Add Candidate workflow enables recruiters and hiring managers to add new candidates to the ATS system for screening, evaluation, and recruitment tracking.

---

## Accessible Roles

| Role | Access |
|--------|--------|
| Admin | Full Access |
| Account Manager | Create & Manage |
| Recruiter | Create & Edit |

---

## Prerequisites

Before adding a candidate:

- User must be logged in.
- Candidate module access must be enabled.
- Resume file should be available.

---

## Workflow

1. Navigate to Candidates.
2. Click **Add Candidate**.
3. Upload resume.
4. Review extracted information.
5. Complete mandatory fields.
6. Save candidate profile.
7. Candidate is added to the pipeline.

---

## Expected Outcome

The ATS creates a candidate profile and makes it available for job matching, interview scheduling, and recruitment tracking.
